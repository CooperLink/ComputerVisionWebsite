Artistic Stylization of Faces

Cooper Link, Devarsi Rawal, Kevin Li, Matt Carroll, Saloni Shah
Fall 2020 CS 4641 Computer Vision: Class Project
Georgia Tech

Problem Statement

The goal of our project is to create an artistically styled interpretation of a portrait that maps important features of an input face onto an input painting without sacrificing the geometric proportions and color palatte of the painting . If no face is detected in the input painting, the input face will be ingrained into the input portrait’s background using texture mapping. If a face is included in the input painting, the input face will replace the detected portrait face using landmark matching while preserving the original portrait’s artistic style and geometric proportions.

For instance, if a user inputs their face and Van Gogh’s Starry Night, the expected output image would have the user’s face ingrained in the overall painting’s background, as there are no faces in Starry Night. However, if a user inputs Da Vinci’s Mona Lisa instead of Starry Night, Lisa’s face would be replaced with an altered version of the user’s face that matches the artist’s style and facial dimensions.

Approach

In order to achieve our goal of artistically mapping a painter’s style onto an input human face, we must utilize various techniques. First, to determine if a painting contains a face, we will use the Viola-Jones algorithm [1]. If a face does not exist, we will simply use a texture generation algorithm [2] to map the landscape painting’s style onto the input face. If a face does exist (and we will have to take into account the possibility of multiple faces), we will have to detect facial landmarks. There exists a previous implementation [3] of this in Python using dlib and OpenCV that we will be able to exploit. Our ultimate goal is to produce results that appear similar to those seen in The Face of Art project [4]. To achieve this, we want to preserve the geometric style of the painting by warping the human source image to match the geometric characteristics of the portrait painting in relation to the face landmark points. We have two ideas for our warping algorithm:

  1. Non-parametric warping using triangular affine matrices [5]
  2. Moving Least Squares algorithm [6] to average the image

To map the artistic styles from image to image, we can go two ways: map the human face onto the painting, or vice versa. Either way, we can implement one of two approaches: Poisson blending [7] or weight masks based on facial feature segmentation. For weight masks, we can simply create a polygon based on the landmark features found previously (the points have ranges assigned to each region) and map those regions over using different weights for different features. The output will then result in an artistically styled face that preserves the original geometry of the painting.



Experiments and Results

With regards to our experimental setup, as previously stated, we want to limit training and creating from scratch our neural networks as it can be costly and highly challenging given our timeframe. We can stay within this constraint by using a pre-trained model for 81-point facial landmark detection [3]. Besides this model, we do not intend on using any other neural networks or deep learning techniques. The datasets we will use to test our methodology are mainly inspired by The Face of Art project [4]; we will utilize the Artistic-Faces dataset [8] offered by IDC Herzliya and the IBUG 300 Faces in the Wild dataset [9] implicitly via the face landmark model. We will also utilize the Best Artworks of All Time dataset [10] in order to test our texture mapping algorithm on landscape paintings. Our implementation will draw inspiration from many sources and exploit some previous code. We will use the aforementioned 81-point face landmark model, a face warping algorithm implemented using dlib [11], a Moving Least Squared algorithm [12], and a Poisson Image editing algorithm [13]. With that given, we will have to implement our own form of Viola-Jones to detect faces in portraits, a texture mapping algorithm for landscape paintings, a style-transfer algorithm using facial feature weight masks, and some small changes to the pre-existing code to make it cooperate with our code. The experiments we will perform are as follows:

  • Test with landscape paintings
  • Test with portrait paintings
  • Test with portrait/landscape paintings with two or more faces
  • Measure effectiveness of mapping artistic style of painting onto input human image
  • Measure effectiveness of blending input human image into the painting itself
  • Measure effectiveness using varying parameter values and weights

We will verify the outcomes of these experiments by qualitatively comparing them to the outputs generated by The Face of Art style transfer implementation.
